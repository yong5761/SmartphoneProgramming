1. 게임 컨셉 및 핵심 메카닉
  High Concept
-2011년도 매우 유명했던 "레이디버그"라는 슈팅 게임을 모티브로 하여 유사한 어플리케이션을 구현한다, 위에서 부터 내려오는 적 오브젝트를 피해 여러 아이템들을 먹어가며 최대한 오래 생존하는것이 목표이다
   핵심 메카닉
-4방향 이동: 기존의 게임인 레이디버그는 핸드폰 기기의 기울기를 통해 플레이어를 조종하지만 수업 내용을 통해 이를 구현하기에는 어려워 보여 터치 및 드래그로 플레이어를 조종하는것을 목표로 한다
-적 오브젝트 소환: 기존의 게임과 유사하게 화면의 상단에서 랜덤하게 적대 오브젝트들을 스폰한다.
-회피 및 생존: 기존의 게임과 유사하게 적 오브젝트에 닿는 순간 게임오버 처리를 하며 닿지않고 오래 돌아다닐 수록 큰 점수를 부여한다.
-아이템: 기존의 게임과 유사하게 중간중간 아이템들을 소환하여 플레이어가 이 아이템에 닿으면 해당 효과를 획득하고 게임 진행에 큰 도움을 부여한다.


2. 개발범위
   개발 주요 요소
-플레이어 캐릭터: 기존의 게임처럼 플레이어에 해당하는 무당벌레 캐릭터를 하나 구현한다, 이는 드래그로 조종이 가능하다. 또한 닿으면 탈락하는 적대 오브젝트들 소환도 구현 하여야 한다.
-아이템: 기존의 게임 "레이디버그"에서는 플레이어가 닿으면서 발동하는 여러 아이템들이 있다. 대표적으로 장판이 깔리며 해당 범위에 닿는 적대 오브젝트를 소멸시키거나, 플레이어 주변에 보호막이 생기며 해당 범위에 닿는 적대 오브젝트를 제거하는 등 여러 아이템이 있고 이러한 것들을 구현하는것이 목표이다
-점수: 기존의 게임과 유사하게 오래 살아남거나, 적대 오브젝트를 아이템을 통해 제거할수록 일정 점수가 증가하는 것을 구현할것이다.
-여러 레이아웃: 기존의 게임은 설정, 점수판, 아이템 설명 등 여러 창이 존재한다. 본인 또한 여러 메뉴 버튼과 이를 통해 갈수 있는 해당 새로운 창을 구현할 것이다.


3. 예상 게임 실행 흐름
   시작 화면
-게임을 시작하면 기본적인 배경화면에 여러 버튼을 배치한다. 게임 시작, 설정(소리),아이템 종류 등이 대표적이다
-게임 시작을 누르면 위에서 계획한 인 게임 장면들이 실행된다.
-설정 버튼을 누르면 소리를 설정할 수 있는 시스템이 나타난다.
-아이템 종류 버튼을 누르면 인 게임 내 존재하는 아이템들의 설명이 나타난다.
